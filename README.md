# Lista interativa de estabelecimentos que oferecem serviços especializados para vítimas de violência sexual

Esse repositório contém:

1. Um dataset com municípios e nomes de estabelecimentos que oferecem atendimento à vítimas de violência sexual, além de outros dados.
2. Um notebook que tem como objetivo criar um mecanismo de busca por estado e município de estabelecimentos que oferecem serviços especializados para vítimas de violência sexual. Pretende-se contribuir para que as vítimas tenham acesso a essa informação e consigam buscar atendimento de maneira mais célere e facilitada.
